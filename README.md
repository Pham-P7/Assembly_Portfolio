# Assembly_Portfolio
# Important Information: <br />
## How to compile in MASM: <br />
  1. enter in terminal -> ml /c /coff "filename.ASM" <br />
  2. enter in terminal -> link /SUBSYSTEM:CONSOLE "filename.obj" <br />
